htmlToTelegram(message, args) → {String}

htmlToTelegram is the function to be used when you have some properties inside some of the valid tags. For Telegram it is not valid to have attributes inside of HTML tags. So you will have to declare all attributes in a JSON.
Parameters:
Name Type Description
message String unformatted HTML text
args Array array of strings with the attributes inside the bold, italic, links tags
Source:
  • index.js, line 32
Returns:
formatted string compatible with Telegram
Type
String
Example
// returns <b>Hello, World!</b><a href="yourwbesite.com">click here</a>
htmlToTelegram('<div><b class="your_class">Hello, World!</b></div><a href="yourwbesite.com">click here</a>', ['class="your_class"']);
